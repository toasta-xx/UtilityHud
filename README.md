# UtilityHUD
- A simple HUD overlay that displays information in-game.

# Features [NEW]
- Speed Display - Shows your current movement speed and maximum speed (current SpeedCap)
- FPS Counter - Displays your current FPS
- Session Timer - Tracks how long you've been playing
- Player Count - lists the number of players in the current room
- Ping Display - Shows your network latency
- Music controller - press right secondary to enable control mode, left trig = prev song, right trig = next song, right primary = pause/play
